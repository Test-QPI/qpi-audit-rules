# Documents Timeline Evidence Rules

These rules govern document timeline fields.

---

## Documents_Timeline_Explicit_YN

YES requires:
- A clearly time‑bound statement
- A date, deadline, or timeframe

Examples:
- “By Friday”
- “Before your start date”
- “Within the next 48 hours”

NON‑examples:
- “When you can”
- “As long as you have a plan”
- “Soon”

---

## Documents_Timeline_Agreed_YN

YES requires:
- An explicit timeline, AND
- Student verbal agreement or acknowledgment

Examples:
- “Yes, that works”
- “I can do that”
- “Sounds good”

---

## Defaults

- If no timeline → Explicit = NO, Agreed = NO
- If timeline stated but no agreement → Explicit = YES, Agreed = NO
