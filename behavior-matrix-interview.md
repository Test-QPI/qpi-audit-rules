# Behavior Matrix — Interview Calls

This matrix evaluates qualitative behaviors during Interview calls.

---

## Behavior Categories

Each category scores 1, 3, or 5:

### Rapport
- 1: Minimal connection
- 3: Professional engagement
- 5: Strong personal rapport

### Empathy
- 1: Ignored concerns
- 3: Acknowledged concerns
- 5: Validated and supported concerns

### Control
- 1: Disorganized or reactive
- 3: Structured but flexible
- 5: Confident call leadership

---

## Evidence Rules

- Evidence must be explicit
- Default to 3 if mixed signals
- Score the **lowest sustained behavior**
