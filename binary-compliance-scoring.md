# Binary Compliance Scoring Rules

---

## Calculation

Binary_Compliance_Rate =
(Binary_Items_Met ÷ Binary_Items_Applicable)

Binary_Compliance_Score =
(Binary_Compliance_Rate × 40)

---

## Rounding Rules

- Score is rounded to the nearest whole number
- Maximum Binary_Compliance_Score = 40
- Binary_Items_Met MUST NOT exceed Binary_Items_Applicable

---

## Defaults

- If no binary items apply, score = 0
- If applicability is ambiguous, default to checklist rules
