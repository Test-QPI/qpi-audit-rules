# Tonality Scoring

This document defines the numeric scoring for tone.

---

## Point Mapping

- Level A – Exceptional → 5 points
- Level B – Positive → 4 points
- Level C – Neutral → 3 points
- Level D – Negative → 1 point
- Level E – Detrimental → 0 points

---

## Rules

- Tonality_Score MUST match Tonality_Level mapping
- Score cap = 5
- Defaults to 0 only if tone is detrimental or missing
